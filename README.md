# Resume Skill Gap Analyzer

## Overview

The Resume Skill Gap Analyzer is an AI-powered web application that analyzes a candidate's resume against a selected job role to identify missing skills and provide personalized recommendations. The system uses Natural Language Processing (NLP) techniques to compare resume content with predefined job requirements.

## Features

- Upload or paste resume text
- Select a target job role
- Automatic skill extraction
- Resume and job description comparison
- Missing skill identification
- Skill match percentage
- Personalized recommendations
- Download analysis report

## Technologies Used

- Python
- Flask
- Natural Language Processing (NLP)
- Pandas
- Scikit-learn
- HTML
- CSS
- Bootstrap
- JavaScript

## Installation

1. Clone this repository

```bash
git clone https://github.com/yourusername/Resume-Skill-Gap-Analyzer.git
```

2. Install the required packages

```bash
pip install -r requirements.txt
```

3. Run the application

```bash
python app.py
```

4. Open your browser and visit

```
http://127.0.0.1:5000
```
